This is a [CMS](https://cms.com) application project bootstrapped with [`create-cms-app`](https://www.npmjs.com/package/create-cms-app).

## Getting Started

First, authenticate to your workspace:

```bash
yarn auth
```

Then, install this app to your workspace:

```bash
yarn sync
```

Open your CMS instance and go to `/settings/applications` section to see the result.
