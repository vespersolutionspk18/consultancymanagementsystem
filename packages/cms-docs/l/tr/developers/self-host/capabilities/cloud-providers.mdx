## Kubernetes'i Terraform ve Manifests ile kullanma

Kubernetes dağıtımı için topluluk liderliğindeki dokümantasyona [buradan](https://github.com/cmshq/cms/tree/main/packages/cms-docker/k8s) erişebilirsiniz.

### Coolify

Deploy CMS on servers using Coolify. (Coolify üzerinde resmi görüntü yakında mevcut olacak)

[Coolify dokümantasyonu](https://coolify.io/docs/get-started/introduction)

### EasyPanel

Deploy CMS on EasyPanel with the community maintained template below.

[EasyPanel'de Dağıt](https://easypanel.io/docs/templates/cms)

### Elest.io

Deploy CMS on servers with Elest.io using link below.

[Elest.io'da Dağıt](https://elest.io/open-source/cms)

### CMS on Railway

Deploy CMS on Railway with the community maintained template below.

[![Railway'de Dağıt](https://railway.com/button.svg)](https://railway.com/deploy/nAL3hA)

### Sealos'ta CMS

Aşağıdaki topluluk tarafından bakımı yapılan şablonla CMS'yi Sealos'ta dağıtın.

[![Sealos'ta Dağıt](https://sealos.io/Deploy-on-Sealos.svg)](https://sealos.io/products/app-store/cms)
